JSONдёҺGOB

еҸ‘иЎЁдәҺ 2023-12-18 жӣҙж–°дәҺ 2024-03-15 еҲҶзұ»дәҺ еҹәзЎҖзҹҘиҜҶ Disqusпјҡ

еңЁ Go иҜӯиЁҖдёӯпјҢJSONпјҲJavaScript Object Notationпјүе’Ң GOBпјҲGo BinaryпјүжҳҜдёӨз§Қеёёз”Ёзҡ„еәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–ж•°жҚ®зҡ„ж јејҸгҖӮencoding/jsonжғіеҝ…еӨ§е®¶йғҪдёҚйҷҢз”ҹпјҢencoding/gobеңЁеёёи§„ golang зҡ„ејҖеҸ‘йЎ№зӣ®дёӯдҪҝз”Ёйў‘зҺҮе°ұдҪҺдәҶеҫҲеӨҡпјҢдёӨиҖ…зҡ„еҢәеҲ«жҳҜд»Җд№ҲпјҹдҪҝз”ЁеңәжҷҜеҸҲжҳҜд»Җд№ҲпјҹдёҖиө·жқҘеӯҰд№ дёӢгҖӮ

JSON дёҺ GOB зҡ„еҢәеҲ«

еҜ№жҜ”з»ҙеәҰ JSON GOB
еҸҜиҜ»жҖ§ JSON жҳҜдёҖз§Қж–Үжң¬ж јејҸпјҢжҳ“дәҺйҳ…иҜ»е’ҢзҗҶи§ЈгҖӮе®ғдҪҝз”Ёй”®еҖјеҜ№зҡ„еҪўејҸиЎЁзӨәж•°жҚ®пјҢе№¶дҪҝз”ЁеӨ§жӢ¬еҸ·е’Ңж–№жӢ¬еҸ·жқҘиЎЁзӨәеҜ№иұЎе’Ңж•°з»„гҖӮ GOB жҳҜдёҖз§ҚдәҢиҝӣеҲ¶ж јејҸпјҢдёҚеҸҜиҜ»пјҢдё»иҰҒз”ЁдәҺеңЁ Go зЁӢеәҸд№Ӣй—ҙй«ҳж•Ҳең°дј иҫ“е’ҢеӯҳеӮЁж•°жҚ®гҖӮ
е…је®№жҖ§ JSON жҳҜдёҖз§ҚйҖҡз”Ёзҡ„ж•°жҚ®дәӨжҚўж јејҸпјҢеҸҜд»Ҙиў«еӨҡз§Қзј–зЁӢиҜӯиЁҖи§Јжһҗе’Ңз”ҹжҲҗгҖӮ GOB жҳҜ Go иҜӯиЁҖзү№жңүзҡ„ж јејҸпјҢеҸӘиғҪеңЁ Go зЁӢеәҸд№Ӣй—ҙиҝӣиЎҢеәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–гҖӮ
ж•°жҚ®зұ»еһӢж”ҜжҢҒ JSON ж”ҜжҢҒеҹәжң¬ж•°жҚ®зұ»еһӢпјҲеҰӮеӯ—з¬ҰдёІгҖҒж•°еӯ—гҖҒеёғе°”еҖјпјүгҖҒж•°з»„гҖҒеҜ№иұЎе’Ң null еҖјгҖӮ GOB ж”ҜжҢҒ Go иҜӯиЁҖзҡ„жүҖжңүж•°жҚ®зұ»еһӢпјҢеҢ…жӢ¬з»“жһ„дҪ“гҖҒеҲҮзүҮгҖҒжҳ е°„гҖҒйҖҡйҒ“зӯүгҖӮ
еәҸеҲ—еҢ–ж•ҲзҺҮ JSON зҡ„еәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–зӣёеҜ№иҫғж…ўпјҢеҚ з”ЁиҫғеӨҡзҡ„еӯҳеӮЁз©әй—ҙе’ҢзҪ‘з»ңеёҰе®ҪгҖӮ з”ұдәҺ GOB жҳҜдәҢиҝӣеҲ¶ж јејҸпјҢзӣёеҜ№дәҺ JSONпјҢе®ғеңЁеәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–иҝҮзЁӢдёӯжӣҙй«ҳж•ҲпјҢеҚ з”Ёжӣҙе°‘зҡ„еӯҳеӮЁз©әй—ҙе’ҢзҪ‘з»ңеёҰе®ҪгҖӮ
и·ЁиҜӯиЁҖж”ҜжҢҒ з”ұдәҺ JSON жҳҜдёҖз§ҚйҖҡз”Ёж јејҸпјҢеҸҜд»Ҙиў«еӨҡз§Қзј–зЁӢиҜӯиЁҖи§Јжһҗе’Ңз”ҹжҲҗпјҢеӣ жӯӨеңЁдёҚеҗҢиҜӯиЁҖд№Ӣй—ҙиҝӣиЎҢж•°жҚ®дәӨжҚўжӣҙж–№дҫҝгҖӮ GOB еҸӘиғҪеңЁ Go иҜӯиЁҖзЁӢеәҸд№Ӣй—ҙдҪҝз”ЁпјҢдёҚйҖӮз”ЁдәҺи·ЁиҜӯиЁҖзҡ„ж•°жҚ®дәӨжҚўгҖӮ

жҖ»з»“пјҡJSON жҳҜдёҖз§ҚйҖҡз”Ёзҡ„гҖҒеҸҜиҜ»жҖ§ејәзҡ„ж•°жҚ®дәӨжҚўж јејҸпјҢйҖӮз”ЁдәҺи·ЁиҜӯиЁҖзҡ„ж•°жҚ®дәӨжҚўгҖӮиҖҢ GOB жҳҜ Go иҜӯиЁҖзү№жңүзҡ„гҖҒй«ҳж•Ҳзҡ„дәҢиҝӣеҲ¶ж јејҸпјҢйҖӮз”ЁдәҺеңЁ Go зЁӢеәҸд№Ӣй—ҙиҝӣиЎҢж•°жҚ®дј иҫ“е’ҢеӯҳеӮЁгҖӮйҖүжӢ©дҪҝз”Ёе“Әз§Қж јејҸеҸ–еҶідәҺдҪ зҡ„е…·дҪ“йңҖжұӮе’ҢдҪҝз”ЁеңәжҷҜгҖӮ

GOB е…·дҪ“дҪҝз”ЁеңәжҷҜ

  1. Go зЁӢеәҸд№Ӣй—ҙзҡ„йҖҡдҝЎпјҡеҰӮжһңдҪ жңүеӨҡдёӘ Go зЁӢеәҸд№Ӣй—ҙйңҖиҰҒиҝӣиЎҢж•°жҚ®дәӨжҚўпјҢдҪҝз”Ё GOB еҸҜд»Ҙе®һзҺ°й«ҳж•Ҳзҡ„еәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–пјҢеҮҸе°‘ж•°жҚ®дј иҫ“зҡ„ејҖй”ҖгҖӮ

  2. ж•°жҚ®жҢҒд№…еҢ–пјҡеҰӮжһңдҪ йңҖиҰҒе°Ҷ Go зЁӢеәҸдёӯзҡ„ж•°жҚ®жҢҒд№…еҢ–еҲ°зЈҒзӣҳжҲ–ж•°жҚ®еә“дёӯпјҢдҪҝз”Ё GOB еҸҜд»Ҙе°Ҷж•°жҚ®д»ҘдәҢиҝӣеҲ¶ж јејҸиҝӣиЎҢеӯҳеӮЁпјҢеҚ з”Ёжӣҙе°‘зҡ„еӯҳеӮЁз©әй—ҙпјҢе№¶дё”иҜ»еҶҷйҖҹеәҰжӣҙеҝ«гҖӮ

  3. зј“еӯҳж•°жҚ®пјҡеҰӮжһңдҪ йңҖиҰҒе°Ҷж•°жҚ®зј“еӯҳеңЁеҶ…еӯҳдёӯпјҢд»ҘжҸҗй«ҳиҜ»еҸ–йҖҹеәҰпјҢдҪҝз”Ё GOB еҸҜд»Ҙе°Ҷж•°жҚ®д»ҘдәҢиҝӣеҲ¶ж јејҸиҝӣиЎҢеәҸеҲ—еҢ–пјҢе№¶еңЁйңҖиҰҒж—¶еҝ«йҖҹеҸҚеәҸеҲ—еҢ–пјҢжҸҗй«ҳзј“еӯҳзҡ„ж•ҲзҺҮгҖӮ

  4. еҲҶеёғејҸзі»з»ҹпјҡеҰӮжһңдҪ жӯЈеңЁжһ„е»әеҲҶеёғејҸзі»з»ҹпјҢйңҖиҰҒеңЁдёҚеҗҢзҡ„иҠӮзӮ№д№Ӣй—ҙдј иҫ“ж•°жҚ®пјҢдҪҝз”Ё GOB еҸҜд»Ҙе®һзҺ°й«ҳж•Ҳзҡ„ж•°жҚ®дј иҫ“пјҢеҮҸе°‘зҪ‘з»ңеёҰе®Ҫзҡ„еҚ з”ЁгҖӮ

  5. RPCпјҲиҝңзЁӢиҝҮзЁӢи°ғз”ЁпјүпјҡеҰӮжһңдҪ дҪҝз”Ё Go иҜӯиЁҖзҡ„ RPC жЎҶжһ¶иҝӣиЎҢиҝңзЁӢиҝҮзЁӢи°ғз”ЁпјҢGOB жҳҜй»ҳи®Өзҡ„еәҸеҲ—еҢ–ж јејҸпјҢеҸҜд»Ҙж–№дҫҝең°иҝӣиЎҢж•°жҚ®дј иҫ“е’Ңи°ғз”ЁгҖӮ

жҖ»з»“пјҡGOB йҖӮз”ЁдәҺеңЁ Go зЁӢеәҸд№Ӣй—ҙй«ҳж•Ҳең°дј иҫ“е’ҢеӯҳеӮЁж•°жҚ®зҡ„еңәжҷҜпјҢзү№еҲ«жҳҜеңЁйңҖиҰҒй«ҳжҖ§иғҪе’ҢдҪҺејҖй”Җзҡ„жғ…еҶөдёӢгҖӮ
