mysql binlog 基础知识

发表于 2025-08-12 分类于 存储/缓存 Disqus：

在 MySQL 数据库中，二进制日志（binary log，通常简称为 binlog）是用于记录所有更改数据库内容的事件的日志文件。它可以用来进行数据恢复、数据复制和审计等。

以下是有关 MySQL binlog 的一些基本概念和使用示例。

1. 启用二进制日志

在使用 binlog 之前，需要确保它在 MySQL 配置文件中启用。可以在 MySQL 的配置文件 my.cnf（或 my.ini）中设置以下参数：```ini
[mysqld]
log-bin=mysql-bin

如果返回值为 ON，则表示 binlog 已启用。
